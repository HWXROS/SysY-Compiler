## 两种使用 Koopa 的方式
### 方式一：输出 IR 字符串（当前做法）
```
// 你的编译器输出字符串
std::cout << "fun @main(): i32 {\n%entry:\n  ret 0\n}\n";

// koopa 库解析字符串
koopa_parse_from_string(ir_string, &program);
```
```
你的编译器                    koopa 库
    │                            │
    │  输出 IR 字符串            │
    │  "fun @main()..."          │
    │ ─────────────────────────► │
    │                            │  解析字符串
    │                            │  构建 IR 对象
```
### 方式二：直接调用 Koopa API（更高效）
```
// 直接用 koopa API 构建 IR，不需要字符串
koopa_raw_function_data_t func;
func.name = "main";
func.return_type = koopa_int32_type();

koopa_raw_basic_block_data_t bb;
bb.name = "entry";

koopa_raw_return_inst_data_t ret;
ret.value = koopa_int_const(0);

// 直接构建，无需解析
```

## 对比
方式 IR 结构 流程 
方式三 （你现在） 自己定义 AST → 自己的IR → 字符串 
方式四 自己定义 + koopa定义 AST → 自己的IR → raw_program → libkoopa 
直接API koopa定义 AST → koopa IR → libkoopa
