1 Flex 默认需要你实现一个 yywrap() 函数，用于 “读完一个文件后，要不要继续读下一个文件”。
    noyywrap = 关掉这个功能，不用写 yywrap 函数，单文件解析够用，编译不报错。
    %option noinput
2 关掉 Flex 内置的 input() 底层读字符函数。
    我们自己不手动读字符、不靠底层裸读，禁用即可，消除警告 / 依赖。
    %option nounput
3 关掉 Flex 内置的 unput() 回退字符函数（把字符塞回流）。
    你的词法规则里没有用到回退字符，禁用，去掉冗余、消警告。\

│ %code requires { 头文件 库}              │  不受 using namespace std 影响
├─────────────────────────────────────────┤
│ %{  // 声明 lexer 函数和错误处理函数        │
│   ...                                   │

│ %}                                   │  │
├──────────────────────────────────────│──┤
│ %parse-param { ... }                 │  │  不在作用域内！
│ %union { ... }                       │  │
│ %token ...                           │  │
├──────────────────────────────────────│──┤
│ %%                                    │  │
│ 规则部分                              ←┘  │  在作用域内
│ %%                                      │
└─────────────────────────────────────────┘

Flex (lexer)                    Bison (parser)
    │                                │
    │  yylval.str_val = "main"       │
    │  return IDENT ─────────────────┼──→ parser 收到 IDENT token
    │                                │     并从 yylval.str_val 获取值


main.cpp                     Flex/Bison 生成的代码
    │                              │
    │  yyin = fopen(input, "r")    │
    │  ───────────────────────────→│  yyin 指向文件/ lexer 从文件读取
    │                              │
    │  yyparse(ast)                │
    │  ───────────────────────────→│  parser 调用 lexer()/ yylex()
    │                              │  解析完成后设置 ast
    |                              |  yylex() 内部:
    │                              │    while ((c = getc(yyin)) != EOF)
    │                              │      // 从 yyin 读取字符
    │  ←───────────────────────────│
    │                              │
    │  cout << *ast                │  输出 AST
